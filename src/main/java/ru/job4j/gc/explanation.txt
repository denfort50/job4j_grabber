1. Я создал модель данных User, у которого 3 поля String, 1 поле int и 2 поля float.
2. 5 объектов User занимают 420 байт или 0,000401 мегабайт (String = 24 байта, int/float = 4 байта).
3. Объект состоит из заголовка, примитивных типов, ссылочных типов и выравнивания для кратности 8 байтам.
В 32-х разрядной системе размер заголовка — 8 байт. Это и будет размер пустого объекта без полей.
4. Установил ключи -Xms2m -Xmx4m. При запуске main увидел, что до создания объектов было меньше свободной памяти,
чем после, следовательно, сборщик мусора сделал свою работу.
